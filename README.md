## 如何使用

```bash
   yum -y install wget screen
   wget https://github.com/movoin/slacker-devel/archive/master.zip
   tar xzf master.gz
   cd master
   chmod +x install.sh
   screen -S slacker
   # 使用默认配置安装
   ./install.sh
```
